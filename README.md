# meta-brief-generator
Statischer Generator für ein Forderungsschreiben (Meta/Instagram) – läuft lokal im Browser, ohne Datenerfassung.

## Datenschutz
- Keine Speicherung
- Keine Übermittlung an Server
- Kein Tracking

## Nutzung
1. Seite öffnen
2. Felder ausfüllen
3. Text kopieren oder als PDF speichern (Browser: Drucken → Als PDF speichern)
4. E-Mail optional vorbefüllen

Keine Rechtsberatung. Nutzung auf eigene Verantwortung.
